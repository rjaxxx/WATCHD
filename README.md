<h1>WATCHD</h1>

<h2>My goal</h2>
My project, Watchd, will be a website that will allow users to log, rate, and add movies and tv shows to their watchlist. I will be focused on making an engaging website, with responsive and accessible code, and ensuring that my design choices ethically support my users. My primary end users are teenagers aged 13 to 18, particularly those using mobile devices and expecting fast visually engaging experiences, which means that they will need my website to be aesthetically pleasing and responsive, as well as mobile friendly. My secondary end users are anyone else who might view the site, like users with disabilities or different accessibility needs, or users of different ages, which means that I need to account for a large majority of users from different backgrounds.

<h2>Requirements</h2>
To run this project, you will need a .env file, with a SECRET_KEY, and a TMDB_API_KEY. 
<ul>
  <li>The secret key can be whatever you choose (for best security a long string of numbers and letters is recommended).</li> 
  <li>The TMDB API key must be a valid API key, more information on how to get one can be found <a href="https://developer.themoviedb.org/docs/getting-started">here</a>.</li>
</ul>
